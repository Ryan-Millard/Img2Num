\doxysubsection{Detailed Description}
Core image processing functions for img2num project. 

This file declares functions for image manipulation, clustering, filtering, and conversion to SVG. Functions operate on raw image buffers (uint8\+\_\+t\texorpdfstring{$\ast$}{*}).
