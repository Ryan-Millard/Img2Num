# 🔍 Lint Scripts

URL: https://img2num.dev/docs/internal/scripts/lint-scripts

## `npm run lint`

Runs ESLint on the entire project.

```bash
eslint .
```

## `npm run lint:fix`

Automatically fixes issues where possible:

```bash
eslint . --fix
```

## `npm run lint:style`

Checks EditorConfig formatting rules:

```bash
editorconfig-checker
```
