\chapter{Img2\+Num Python Bindings}
\hypertarget{index}{}\label{index}\index{Img2Num Python Bindings@{Img2Num Python Bindings}}
\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
The {\bfseries{Img2\+Num Python Bindings}} provide a Python interface to the core Img2\+Num image processing library via \href{https://github.com/pybind/pybind11}{\texttt{ pybind11}} (also see \href{https://pybind11.readthedocs.io/en/stable/}{\texttt{ their docs}}). These bindings wrap the underlying C++ functionality, exposing image processing operations such as filtering, clustering, thresholding, and SVG conversion to Python code.

All functions operate on {\ttfamily numpy.\+ndarray} buffers and return new image data, making them easy to integrate into Python-\/based image processing pipelines.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Key Features}{Key Features}}\label{index_features_sec}
The Python bindings expose the following core capabilities\+:
\begin{DoxyItemize}
\item {\bfseries{FFT-\/based Gaussian blur}} — Fast frequency-\/domain blurring
\item {\bfseries{Image inversion}} — Invert pixel values
\item {\bfseries{Thresholding}} — Standard and black-\/thresholding operations
\item {\bfseries{K-\/means clustering}} — Pixel clustering for color quantization
\item {\bfseries{Bilateral filtering}} — Edge-\/preserving smoothing
\item {\bfseries{Label-\/to-\/\+SVG conversion}} — Convert labeled images to SVG strings
\item {\bfseries{Image-\/to-\/\+SVG conversion}} — Full image vectorization with configurable parameters
\item {\bfseries{Image\+To\+Svg\+Config}} — Python-\/accessible configuration for SVG generation
\end{DoxyItemize}\hypertarget{index_building_sec}{}\doxysection{\texorpdfstring{Building the Python Bindings}{Building the Python Bindings}}\label{index_building_sec}
