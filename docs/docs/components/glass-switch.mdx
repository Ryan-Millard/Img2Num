## GlassSwitch

`GlassSwitch` is a reusable, accessible toggle switch component built using the global `glass` styles.

It is a **controlled component**, meaning the parent component is responsible for managing its state.
`GlassSwitch` focuses purely on presentation and interaction and does not contain any business or theme logic.

This makes it suitable for use cases such as:
- Theme toggles
- Feature flags
- Settings switches
- Any boolean on/off state

---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `isOn` | `boolean` | Yes | — | Whether the switch is toggled on |
| `onChange` | `function` | Yes | — | Callback function fired when the switch is clicked |
| `ariaLabel` | `string` | Yes | — | Accessibility label for the switch |
| `thumbContent` | `ReactNode` | No | `undefined` | Content to display inside the switch thumb |
| `disabled` | `boolean` | No | `false` | Whether the switch is disabled |
| `size` | `string` | No | `'medium'` | Size of the switch (`'small'`, `'medium'`, `'large'`) |

---

## Basic Usage

```jsx
import { useState } from 'react';
import { Moon, Sun } from 'lucide-react';
import GlassSwitch from '@/components/GlassSwitch';

const Example = () => {
  const [isOn, setIsOn] = useState(false);

  return (
    <GlassSwitch
      isOn={isOn}
      onChange={() => setIsOn(prev => !prev)}
      thumbContent={isOn ? <Moon /> : <Sun />}
      ariaLabel="Toggle feature"
    />
  );
};
```

## With Optional Props

```jsx
import { useState } from 'react';
import GlassSwitch from '@/components/GlassSwitch';

const ExampleWithOptions = () => {
  const [isOn, setIsOn] = useState(false);

  return (
    <GlassSwitch
      isOn={isOn}
      onChange={() => setIsOn(prev => !prev)}
      ariaLabel="Large switch"
      size="large"
      disabled={false}
    />
  );
};
```
